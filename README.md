# **Comandos basicos Git**
* git init
* git status
* git add
* git add . (all files)
* git commit (-a, -m) (update)
* git push (send)
* git pull (receive)
* git rm (remove)
* git log
* git mv (move and rename)
------------------------

# **Git VsCode**
* M = Modified
* U = Untracked
* A = Add to project
* R = Rename
